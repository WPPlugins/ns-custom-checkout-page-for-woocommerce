This plugin help to customize your checkout field for WooCommerce, no code required!

== Description ==
This plugin allow you to show/hide field in WooCommerce checkout page.
Page option will appear in your WordPress backend menu, and here you can choose if show or hide every single filed.
You only have to select checkbox that you want to hide and save it!
Checkout field selected will be hidden.

**Example** for italian customers: you can use this plugin to **add a VAT field** inside checkout page

PREMIUM FEATURES of custom checkout page:

You can **add custom filed**.
You can also **flag required field**, customise **placeholder** text, **order** field and replace **label** of any fields.

== Installation ==

1. Upload the entire 'ns-custom-checkout-fields' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Custom Checkout' and flag fields to hide
